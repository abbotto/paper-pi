Paper-Pi
-----------
A template for a sturdy R-Pi case which is made from card-stock.<br>
URL: http://o0110o.github.com/paper-pi

Usage
-----
1. Print
2. Cut
3. Score
5. Glue
6. Spray with a polyurethane finish for extra protection
